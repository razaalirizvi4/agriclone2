## Should do work with Google Gemini CLI.
## Steps to implement a feature or update.

# Steps
# 1-Create/open a folder of your name in prompts directory.
# 2-Create new file (MM-DD-YYYY_FeatureName) or open today's working file.
# 3-Write prompt related to feature and which doc file it would impact, make sure to only impact on the area/point of the existing doc feature if partial work is required.
# 4-Gemini will read the prompt file and write/update doc feature file where implementation task are written in detail which can be treated also as documentation as well as tasks for Gemini to implement.
# 5-Read the generated/updated doc feature md file and ensure the updated implementation tasks are correct.
# 6-Make consensus with relevant team mates related to impacted feature doc.
# 7-Gemini will then start to implement it. assist it in the process if it gets stuck following best practice for prompt engineering and coding
# 8-Review implementation and perform developer testing.
# 9-Optional. write unit tests related to the feature.fghj
