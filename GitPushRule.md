Use tortoise git tool.

Steps to Push your code changes:

1- Pull from tortoise git
    -Select Rebase Option.
    -Stash
    -Fast Forward
    -Stash Pop
        (Conflict resolve if any)
2- Commit your changes.
3- Push. (if someone pushed before your 1-Pull and 3-Push, repeat the steps)

-Follow commit message guidelines. Ask if you do not know the guidelines.
